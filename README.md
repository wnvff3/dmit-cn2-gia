## CN2 GIA 是什么，为什么国内用户这么在意

CN2 GIA 是中国电信的国际精品网线路（CTGNet/AS4809），"GIA"是 Global Internet Access 的缩写。它的核心优势是什么？

三个字：优先级高。

普通国际线路共享公网，高峰期就排队，延迟飙升很正常。CN2 GIA 的流量走专用通道，中国电信会优先调度，所以晚上八九点那会儿，别的服务器在跑 300-400ms 甚至断连，CN2 GIA 的机器通常能守住 150-180ms 的正常延迟范围，丢包也少得多。

对于主要用途是科学上网、跨境站建设、国内访问频繁的场景，这条线的意义是真实的，不是营销词。
